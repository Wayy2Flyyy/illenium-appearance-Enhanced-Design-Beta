# illenium-appearance Enhanced Design Beta

**illenium-appearance Enhanced Design** is a premium **W2F-style redesign** of the standard FiveM clothing menu, built with a cleaner layout, smoother navigation, sharper character preview, and a more polished customization flow for serious roleplay servers.

<img width="1672" height="941" alt="illenium-appearance Enhanced Design Preview" src="https://github.com/user-attachments/assets/a0f27058-ea44-46ec-8479-24195c969a21" />

## About

This beta improves the overall clothing menu experience while keeping the familiar foundation of `illenium-appearance`. The goal is to make the UI feel more modern, readable, and professional without making it complicated for players to use.
